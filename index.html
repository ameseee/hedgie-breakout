Save the Hedgie

The goal of the game is to break all the (breakable) blocks on the screen. Blocks can be broken by colliding with the ball. Some more things to know before you play:

  • Hitting each breakable block earns you 10 points.
  • You have three chances, but each time you let the ball drop below the screen, you lose 50 points.
  • There are four levels to beat!
  • Use the mouse OR the left/right arrow keys to control the paddle.
  • Each level has 1-3 powerups. Break a red block to earn 25 points and unlock a powerup. Catch the powerup to earn 25 additional points, and activate a random powerup (fast or slow ball, long or short paddle).
  • Grey, UNBREAKABLE, blocks will start appearing in Level 2. Hitting them earns NO points.
